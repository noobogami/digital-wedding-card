## 📋 Next Steps - Customization

### Step 1: Replace Placeholder Content (5-10 minutes)

Search for `PLACEHOLDER` comments in these files:

1. **src/components/Hero.jsx**
   - Replace couple names
   - Update date and time

2. **src/components/Details.jsx**
   - Update ceremony details
   - Add contact information

3. **src/components/Location.jsx**
   - Add venue address
   - Add Google Maps API key
   - Update coordinates

4. **src/App.jsx**
   - Update footer year

### Step 2: Add Your Music (2 minutes)

```bash
# Place your music file here:
public/music/background-music.mp3
```

### Step 3: Get Google Maps API Key (5 minutes)

1. Visit: https://console.cloud.google.com/
2. Create/select project
3. Enable "Maps Embed API"
4. Create API key
5. Add to Location.jsx

### Step 4: Test Everything (10 minutes)

- [ ] Test on desktop browser
- [ ] Test on mobile device
- [ ] Check all text is correct
- [ ] Verify music plays
- [ ] Verify map shows correctly
- [ ] Test all buttons and links

### Step 5: Deploy (10 minutes)

```bash
# Build for production
npm run build

# Deploy the 'dist' folder to:
# - Vercel (easiest)
# - Netlify
# - GitHub Pages
# - Any web hosting
```
